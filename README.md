# Atzee Landing

Landing page and brand presentation website for the Atzee platform ecosystem.


## Overview

Atzee Landing is the public-facing website for the Atzee ecosystem.

The project is designed to present the Atzee platform vision, product direction, enterprise ecosystem, and business solutions through a modern and scalable frontend experience.



## Main Goals

- Present the Atzee brand identity
- Showcase platform ecosystem
- Introduce enterprise solutions
- Deliver modern landing page experience
- Support future product marketing



## Tech Stack

- React
- TypeScript
- Vite



## Features

- Responsive landing page
- Modern UI layout
- Product showcase sections
- Platform overview
- Enterprise-oriented presentation

---

## Getting Started

### Installation

```
npm install
```

### Development Server
```
npm run dev
```

### Build
```
npm run build
```
